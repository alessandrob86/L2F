# Tecnologie batterie

Tre tecnologie nella gamma [[Batterie]]:

- **AGM** (Absorbent Glass Mat) - linea **Astra**. Massime prestazioni per veicoli con Start&Stop ed elevati assorbimenti elettrici.
- **EFB** (Enhanced Flooded Battery) - linea **Eclipse**. Per lo Start&Stop di base.
- **Standard** - linee **Syncro, Titan, Top Performance**. Affidabilita per impieghi tradizionali.

Nota informativa sopra al catalogo batterie.
