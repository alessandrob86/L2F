# Focus

> 5.000 lm - per fari a parabola

Modello della linea **[[Lampade LED]]** L2F.

Progettata specificamente ed esclusivamente per **fari a parabola** (riflettore), adotta una **Focus Lens integrata** che emula l'effetto lenticolare, massimizzando la luminosita ed evitando dispersioni. Luce fredda da 5000 lumen con proiezione precisa e definita, e durata elevatissima nel tempo.

## Specifiche

| Parametro | Valore |
|---|---|
| Flusso luminoso | 5.000 lm |
| Temperatura colore | 6000K (bianco freddo) |
| Potenza | 30W per lampada |
| Voltaggio | 9-30V DC |
| Chip LED | GC 3570 |
| Raffreddamento | Ventola interna |
| Ottica | Focus Lens integrata |
| Quantita LED | 4/8 per lampada |
| Corrente assorbimento | 2.0A +/-0.2A |
| Utilizzo | Anabbagliante e antinebbia |

## Attacchi a catalogo
H4, H7, H8

- **H8** copre lo stesso socket di **H8 / H9 / H11 / H16** (medesimo innesto).

**Applicazione:** Fari a parabola / riflettore

## Nota compatibilita
⚠️ **Solo per fari a PARABOLA (a riflettore).** La Focus Lens emula l'effetto lenticolare ma la lampada **NON va installata sui proiettori lenticolari** — per quei fari si usano [[LED Nova|Nova]] o [[LED Transition|Transition]]. Verificare lo spazio disponibile nel faro (kit con lente). Alcuni modelli possono richiedere un CANBUS Decoder. Installazione professionale raccomandata.
