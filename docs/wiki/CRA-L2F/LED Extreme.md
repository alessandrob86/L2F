# Extreme

> 12.000 lm - massima potenza

Modello della linea **[[Lampade LED]]** L2F.

Il massimo della gamma in termini di performance e affidabilita: 12000 lumen per una visibilita ottimale in ogni condizione. Il raffreddamento potenziato (ventola da **36mm** e **doppio tubo di rame**) mantiene temperature stabili sostenendo il chip CSP 3570. La centralina **Super CANBUS** in dotazione assicura compatibilita universale senza errori sul cruscotto.

## Specifiche

| Parametro | Valore |
|---|---|
| Flusso luminoso | 12.000 lm |
| Temperatura colore | 6000K (bianco freddo puro) |
| Potenza | 65W per lampada |
| Voltaggio | 9-32V DC |
| Chip LED | CSP 3570 (6*55mil) |
| Raffreddamento | Ventola 36mm + doppio tubo di rame |
| Quantita LED | 4/8 per lampada |
| Corrente assorbimento | 4.8A +/-0.2A |
| Centralina | Super CANBUS Error Free (inclusa) |
| Utilizzo | Anabbagliante, abbagliante, antinebbia |

## Attacchi a catalogo
H1, H4, H7

**Applicazione:** Veicoli leggeri e pesanti

## Nota compatibilita
Il montaggio puo risultare piu complesso: verificare lo spazio nel proiettore. Installazione professionale raccomandata.
