# Nova

> 9.500 lm - per proiettori lenticolari

Modello della linea **[[Lampade LED]]** L2F.

La vera stella della gamma, progettata esclusivamente per **proiettori lenticolari**: il **design a diamante** con angolo luminoso a **360 gradi** ottimizza il flusso per 9500 lumen, senza lasciare angoli ciechi. Chip Premium **G-CR** ad alta efficienza e resistenza al calore.

## Specifiche

| Parametro | Valore |
|---|---|
| Flusso luminoso | 9.500 lm |
| Temperatura colore | 6500K (bianco freddo) |
| Potenza | 60W per lampada |
| Voltaggio | 9-32V DC |
| Chip LED | G-CR (Premium) |
| Angolo luminoso | 360 gradi (design a diamante) |
| Quantita LED | 2 per lampada |
| Corrente assorbimento | 4.6A +/-0.2A |
| Utilizzo | Anabbagliante, abbagliante, antinebbia |
