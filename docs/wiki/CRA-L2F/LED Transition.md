# Transition

> 9.000 lm - sostituzione Xenon

Modello della linea **[[Lampade LED]]** L2F.

Pensata per chi vuole abbandonare lo **Xenon**: sostituzione diretta delle lampade originali **D1S/D2S/D3S/D5S** con 9000 lumen. Si alimenta dal **ballast HID originale** del veicolo, quindi non richiede alcuna modifica all'impianto elettrico (Plug & Play OEM). Chip **SANAN** ad alta affidabilita.

## Specifiche

| Parametro | Valore |
|---|---|
| Flusso luminoso | 9.000 lm |
| Temperatura colore | 6500K (bianco freddo) |
| Potenza | 45W per lampada |
| Alimentazione | Ballast HID/OEM originale |
| Chip LED | SANAN |
| Angolo luminoso | 360 gradi |
| Quantita LED | 4 per lampada |
| Corrente assorbimento | 3.0A +/-0.1A |
| Utilizzo | Anabbagliante (sostituzione Xenon) |

## Attacchi a catalogo
D1S, D2S, D3S, D5S

**Applicazione:** Veicoli con impianto Xenon/HID

## Nota compatibilita
Verificare il modello esatto di lampada Xenon installata prima dell'acquisto. Nessuna modifica elettrica grazie al ballast originale. Installazione professionale raccomandata.
