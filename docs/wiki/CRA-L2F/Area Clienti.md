# Area Clienti

Spazio riservato all'officina sul portale L2F (`/area-clienti`). Si apre cliccando il nome utente nella navbar.

## Funzioni (attive)
- Login/registrazione officina (Supabase Auth); stato in attesa -> attiva.
- Sblocco **prezzi netti** sul [[Catalogo L2F]] (solo officina attiva).
- **Barra [[Cashback]]** animata verso l'obiettivo + modalità Elite.
- Pannello **[[L2F Academy]]**: crediti corso rimasti + link al catalogo [[Corsi]].
- **Storico ordini** espandibili (numero, data, stato, totale, righe con miniatura prodotto).

Account unico condiviso con [[Centro Ricambi Auto|CRA]] -> [[Interconnessioni]] - [[Backend Supabase]]. Gestione lato L2F nel [[Pannello Admin]].
