# Matrix

> 7.000 lm - livello superiore

Modello della linea **[[Lampade LED]]** L2F.

Illuminazione di livello superiore: la luminosita di ben 7000 lumen migliora sensibilmente la percezione della strada, anche grazie alla temperatura colore fredda e brillante. Il chip **CSP 3570** lavora sempre al massimo dell'efficienza grazie alla ventola interna compatta che assicura un raffreddamento costante. Installazione **Plug & Play** senza adattamenti.

## Specifiche

| Parametro | Valore |
|---|---|
| Flusso luminoso | 7.000 lm |
| Temperatura colore | 6500K (bianco freddo brillante) |
| Potenza | 33W per lampada |
| Voltaggio | 9-30V DC |
| Chip LED | CSP 3570 |
| Raffreddamento | Ventola interna + tubo di rame |
| Quantita LED | 4/8 per lampada |
| Corrente assorbimento | 2.55A +/-0.1A |
| Utilizzo | Anabbagliante, abbagliante, antinebbia |

## Attacchi a catalogo
H1, H3, H4, H7

**Applicazione:** Auto, moto e veicoli leggeri (impianto 12V)

## Nota compatibilita
La **H7** è **1:1 con la lampada alogena** (stessa altezza, 51,5 mm) ed estremamente sottile: si installa praticamente ovunque senza ingombri. Alcuni veicoli del Gruppo VAG e Mercedes-Benz possono richiedere un CANBUS Decoder.
