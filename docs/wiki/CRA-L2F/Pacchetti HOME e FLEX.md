# Pacchetti HOME e FLEX

Il modello a servizi per le officine del network L2F. Prezzi +IVA/anno (fonte: Supabase/sito).

## HOME - 590 EUR
Base: kit benvenuto (insegna, abbigliamento, cancelleria), **1 corso di formazione annuale** ([[L2F Academy]], = 1 credito corso), [[Reso a Vista]], accesso [[Cashback|cashback 3%]].

## FLEX (porta a [[Cashback|cashback 5%]])
- **Flex Marketing - 130 EUR**: tutti i servizi Home + marketing personalizzato, social, campagne sponsorizzate.
- **Flex Tech - 990 EUR**: Home + L2F Tech (banca dati tecnica).
- **Flex All Included - 1650 EUR**: pacchetto completo (Home + Tech + Marketing & Social).

Logica: si parte da HOME, si passa a FLEX aggiungendo un servizio. Dall'[[Area Clienti]] il cliente può chiedere **upgrade/downgrade** del pacchetto (richiesta via email, gestita nel [[Pannello Admin]]). Vedi [[E-commerce B2B]].
